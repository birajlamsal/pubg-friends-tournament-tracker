PUBG Friends Tournament Tracker

PUBG Friends Tournament Tracker is an open-source application designed to track match statistics, player performance, and custom tournament standings for private PUBG PC matches played among friends.

The project uses the official PUBG API to fetch match data such as kills, damage, placement, and results, and presents them in a simple leaderboard and analytics format.

              Development Status:
This project is currently in active development. Features, structure, and UI may change as development progresses.
